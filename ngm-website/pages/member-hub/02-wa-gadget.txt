NATIVE WA GADGET — Upcoming events  (the "Coming up" list)
============================================================
Between the 01-top and 03-bottom Custom HTML gadgets, add the native
Wild Apricot "Upcoming events" gadget.

  1. Insert gadget → Upcoming events.
  2. In its settings, set the gadget style to "Style 1" (upcomingEventsStyle001).
  3. Add the CSS class:  ngm-wa-events
     (Gadget settings → Advanced / custom CSS class.)
  4. It will be skinned into the NGM card-list automatically (the styling +
     the badge script live in 01-top / global.css). No other setup needed.

This is the ONLY dynamic part of the page — it shows real upcoming events pulled
live from the calendar. Everything else is static HTML.
