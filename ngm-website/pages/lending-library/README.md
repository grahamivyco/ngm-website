# Lending Library

Members-only page explaining how the Guild's lending library actually works &mdash;
the collection lives in three locked cabinets at MVUUF and is browsed and borrowed
**in person during meetings**. Content comes from the librarian (Janet Rock).

## Wild Apricot setup
1. Create a page at `/lending-library`.
2. Restrict it to members (borrowing is a current-member benefit).
3. Paste `01-top.html` into a Custom HTML gadget. Use `01-top.slim.html` once the
   new Global CSS is live.

## Deliberate content decisions
- **The old catalog spreadsheet is NOT linked.** The librarian confirmed it is out
  of date, and a refreshed catalog is in progress. Publishing the stale list would
  mislead members. There's an HTML comment marking where to add the new catalog
  link once it's ready.
- **No target date is published.** The refreshed catalog was described as a hope,
  not a commitment, so the page says only that it's "in the works."
