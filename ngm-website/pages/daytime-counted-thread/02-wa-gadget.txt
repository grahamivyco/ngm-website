NATIVE WA GADGET — "Upcoming events"
=====================================
Add the native WA "Upcoming events" gadget BETWEEN 01-top.html and 03-bottom.html,
in the SAME layout column.

1. Add gadget -> "Upcoming events".
2. Gadget settings -> "CSS class" -> enter:  ngm-wa-events
3. FILTER it to Daytime Counted Thread & Needlepoint events only.
4. Set the layout ROW background to linen (#F0EAE0).

Event-card styling for .ngm-wa-events is in 01-top.html.
