NATIVE WA GADGET — "Upcoming events"
=====================================
Not pasteable HTML. Add the native WA "Upcoming events" gadget in the
WA editor, BETWEEN 01-top.html and 03-bottom.html, in the SAME layout row.

Setup:
1. Add gadget → "Upcoming events".
2. Select it → gadget settings → "CSS class" → enter:  ngm-wa-events
3. FILTER the gadget to the events relevant to THIS page
   (Stitch-Ins / meetings / special events — see 01-top.html header).
4. Set the layout ROW background to transparent or linen (#F0EAE0).

Tag chips + whole-row click are injected by the [EVENT-TAGS] script
at the bottom of 01-top.html.
